# wims-registration-helper

WIMS 등록/이전 업무를 돕기 위한 웹 서비스 프로젝트입니다.

## 프로젝트 개요
- 프론트엔드: 사용자 화면(회원가입, 로그인, 개체 등록, 양도/양수, 프로필)
- 백엔드: 인증, 재고(개체) 관리, 양도/양수 트랜잭션 처리, DB 영속화
- 목표: PRD 기반으로 실제 동작 가능한 서비스와 포트폴리오 수준의 설계/구현 완성

## 현재 진행 방향
- 초기 단계는 더미 데이터/로컬 개발 환경으로 기능을 빠르게 검증
- 이후 PostgreSQL(AWS) + 마이그레이션 기반으로 전환하여 실제 운영형 구조로 확장
